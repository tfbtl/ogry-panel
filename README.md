# Ogrency Panel

This project is part of the **Ogrency Ecosystem**.

## Purpose

Ogrency Panel is a React-based administrative interface used to manage
and operate Ogrency backend services.

## Architecture

- Frontend: React (Vite)
- Backend: Ogrency Web APIs
- Authentication: Supabase (Auth only)
